Usage:
-------

1) Download and unzip the states-list.zip file.

2) Five files named "index.html, style.css,readme.txt, jquery.js and state.js" will be obtained.

3) Run the index.php file in your web browser to view the script functionality.

Index.html file:
----------------
This file contains the entire functionality of ‘states-list script’.

 
Jquery.js:
-----------

This js file contains all the jquery functions. Do not edit the code inside this file. If any changes are done, then it will affect the functionality of the script.

state.js:
----------

1) This js file is used to list the districts of selected state.

Style.CSS:
----------

1) Style.css contains the 'CSS properties' of the script

2) CSS properties like 'Background, padding, margin, height, width, border' etc., can be modified according to your needs.

Note: Before replacing the stylesheet make sure that the classnames and id names are given properly, to avoid duplication.

How to embed the script in to your webpage?
--------------------------------------------

1) Open the index.html file.

2) Include 
		<script language="Javascript" src="jquery.js"></script>
		<script type="text/JavaScript" src='state.js'></script>
		<link rel="stylesheet" type="text/css" href="style.css">

   in the <head> tag.

3) States names are stored in the array variable 'handles'.

2) Make sure that the jquery.js,state.js and style.css files are placed in appropriate folders.
